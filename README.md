**Description**

This is a project which extends a task manager application by implementing basic CRUD operations.

**Setup Instructions**
1. Remotely connect to the provided EC2 instance, using OpenSSH or PuTTy
2. Make sure backend and frontend are running by entering: 'pm2 status'. if its not running, enter 'pm2 start all'.
3. In your browser, go to the baseURL in the axiosConfig.jsx file.
